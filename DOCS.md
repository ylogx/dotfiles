Docs
====

## Helper
### Track files
Track a new file using:

```sh
chezmoi add ~/.some_config
```

### Edit existing files
```sh
chezmoi edit ~/.zshrc
```

### Apply changes
```shell
chezmoi apply --verbose
```

### See what would change
```sh
chezmoi diff
```

See [chezmoi.io](https://www.chezmoi.io/user-guide/command-overview/) for more.
